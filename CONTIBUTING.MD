# Contributing to VoiceAgentArnab

Thank you for contributing to VoiceAgentArnab!

We welcome:
- Bug reports
- Feature requests
- Documentation improvements
- Code contributions
- Performance optimizations

---

# Getting Started

## Fork the Repository

Click the "Fork" button on GitHub.

---

# Clone Your Fork

```bash
git clone https://github.com/YOUR_USERNAME/VoiceAgentAI.git
```

---

# Create Virtual Environment

```bash
python -m venv venv
```

Activate:

## Windows

```bash
venv\Scripts\activate
```

## Linux / Mac

```bash
source venv/bin/activate
```

---

# Install Dependencies

```bash
pip install -r requirements.txt
```

---

# Run Locally

```bash
python test.py
```

---

# Creating a Pull Request

1. Create a new branch
2. Make your changes
3. Commit changes
4. Push branch
5. Open Pull Request

Example:

```bash
git checkout -b feature/add-whisper-support
```

---

# Reporting Bugs

Please create an Issue including:

- OS
- Python version
- Error logs
- Steps to reproduce

---

# Feature Requests

Feature requests are welcome.

Please explain:
- use case
- expected behavior
- possible implementation

---

# Coding Guidelines

- Keep code modular
- Add comments when necessary
- Avoid breaking existing APIs
- Follow Python best practices
